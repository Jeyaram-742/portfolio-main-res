Zoo Management System

Online Movie Ticket Booking System

I developed an Online Movie Ticket Booking System using PHP and MySQL to facilitate the seamless booking of movie tickets for users. This system provides features such as movie listings, showtimes, seat selection, and secure online payment integration. The front-end interface, designed with HTML, CSS, and Bootstrap, offers a responsive and user-friendly experience, while the back-end, powered by PHP and MySQL, handles user authentication, booking management, and real-time data updates. Graphical Password Authentication System Using CCP

I developed a Graphical Password Authentication System using the Cued Click Points (CCP) method, aimed at enhancing security and usability in user authentication. This system allows users to select specific points on images as their passwords, offering an intuitive and secure alternative to traditional text-based passwords. I utilized HTML, CSS, Bootstrap, and JavaScript for the front-end, and JSP with MySQL for the back-end. The project involved designing a responsive user interface, implementing secure authentication processes, and ensuring robust data protection. This innovative solution effectively demonstrates my skills in web development and understanding of security principles.
